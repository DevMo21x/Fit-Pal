# FitPal

FitPal is a full-stack web application designed to help users track their workout sessions, manage fitness goals, and maintain a healthy lifestyle. This project is developed as part of a semester-long course.

## Features

- User authentication (Sign Up, Sign In, Protected Routes)
- Create, edit, and delete workout sessions
- View workout history and progress
- Responsive design for mobile and desktop

## Tech Stack

- **Frontend**: React, Vite
- **Backend**: Node.js, Express
- **Database**: MongoDB
- **Authentication**: JSON Web Tokens (JWT), bcrypt

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/DevMo21x/Fit-Pal.git
   ```

2. Navigate to the project directory:

   ```bash
   cd Fit-Pal
   ```

3. Install dependencies for both the client and server:

   ```bash
   # Install server dependencies
   npm install

   # Navigate to the client folder and install dependencies
   cd client
   npm install
   ```

4. Start the development server:

   ```bash
   # Start the backend server
   npm run dev

   # Start the frontend server
   cd client
   npm run dev
   ```

5. Open your browser and navigate to `http://localhost:3000`.

## Usage

- Register or log in to your account.
- Create and manage workout sessions.
- Track your progress over time.
